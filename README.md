# 一键文字转换

一个用AI写的简单本地单文件网页工具，用于预览字体并导出透明 PNG、SVG 和可编辑轮廓 DXF 文件。

## 使用方法

1. 双击打开 `文字预览.html`。
2. 拖入或手动选择 `.ttf`、`.otf`、`.woff`、`.woff2` 字体文件。
3. 输入需要转换的文字，调整导出尺寸、字号、字距、颜色和曲线精度。
4. 点击对应按钮导出 PNG、SVG 或 DXF 文件。

## 导出说明

- PNG：透明背景位图，适合预览和普通图片使用。
- SVG：透明背景矢量文件，保留文本形式，适合网页和设计软件继续处理。
- DXF：通过 Canvas 渲染和边缘追踪提取文字轮廓，导出 AutoCAD R12 ASCII DXF，可在 CAD 软件中打开并编辑闭合多段线轮廓。

## DXF 注意事项

- DXF 文件中的文字已经转为轮廓线，不再依赖字体文件。
- 曲线精度数值越小，轮廓越细腻，文件也会更大。
- 导出的 DXF 可用于激光切割、雕刻、CNC 等需要文字轮廓的场景。
- 如需 DWG，可在 AutoCAD 或兼容软件中打开 DXF 后另存为 DWG。
